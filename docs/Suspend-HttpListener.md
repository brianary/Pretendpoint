# Suspend-HttpListener

## SYNOPSIS
Pauses an HTTP listener.

## SYNTAX

```
Suspend-HttpListener [-Listener] <HttpListener> [<CommonParameters>]
```

## DESCRIPTION
{{ Fill in the Description }}

## EXAMPLES

### EXAMPLE 1
```
Suspend-HttpListener $http
```

The $http listener is paused.
